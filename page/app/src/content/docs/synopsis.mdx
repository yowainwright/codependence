## Synopsis

CLI and Node tool that compares a "codependencies" array against `package.json` dependencies, devDependencies, and peerDependencies.

For each dependency: checks if versions are at latest or match specified versions. Returns pass/fail or auto-updates `package.json` files.

Ensures specified dependencies are up-to-date or pinned to specific versionsâ€”giving you full control over dependency management.

**Codependencies:** Project dependencies that must be up-to-date or set to a specific version. When `latest` can't be specified in `package.json`, Codependence ensures the actual latest semver version is set.
