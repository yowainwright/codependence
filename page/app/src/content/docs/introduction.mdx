## Introduction

JavaScript utility to ensure dependencies are up-to-date or match specified versions.

## Synopsis

Checks your "codependencies" array against `package.json` dependencies—verifying each is at latest or matches a specified version. Pass/fail CI or auto-update mismatched dependencies.

## Why use Codependence?

- Manages dependency versions with granular control
- Handles monorepos—checks multiple `package.json` files
- Runs in CI via npm scripts or GitHub Actions
- Integrates with build tools like [Husky](https://github.com/typicode/husky)

## Why not use Codependence?

Skip if you don't need strict version control or prefer manual pinning.
