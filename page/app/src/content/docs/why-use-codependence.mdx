## Why use Codependence?

Manages dependency versions with granular control.

- Fully manages dependencies—no additional tools needed
- Handles monorepos without package manager bias
- Runs immediately via npm scripts or build tools like Husky
- Integrates with CI via npm scripts or GitHub Actions
